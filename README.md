SPRING ORM CRUD OPERATION
========================

A Java-based web application that demonstrates CRUD
(Create, Read, Update, Delete) operations using
Spring MVC, Spring ORM, Hibernate, and Maven.

This project follows the MVC architecture and shows
how database operations are handled using ORM.


FEATURES
--------
- Create, Read, Update, Delete records
- Spring ORM integration with Hibernate
- MVC-based project structure
- Maven dependency management
- JSP-based user interface


TECH STACK
----------
Java
JSP & Servlets
Spring MVC
Spring ORM
Hibernate
Maven
MySQL
Apache Tomcat

HOW TO RUN THE PROJECT
---------------------

PREREQUISITES
- JDK 8 or above
- Maven
- Apache Tomcat
- MySQL
- Eclipse or any Java IDE


DATABASE SETUP
--------------
1. Open MySQL
2. Create database

   CREATE DATABASE springcrud;

3. Update database username and password
   in the Spring configuration file


STEPS
-----
1. Clone the repository

   git clone https://github.com/SamratDesai7/Spring-orm-CURd-Operation.git

2. Import into IDE
   - Open Eclipse
   - Import as Existing Maven Project

3. Build the project

   mvn clean install

4. Deploy the project on Apache Tomcat

5. Open in browser

   http://localhost:8080/Spring-orm-CURd-Operation
